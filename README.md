# 💰 Controle de Finanças Pessoal

Um web app simples, intuitivo e responsivo para controle de gastos mensais.  
Criado com foco em facilidade de uso no dia a dia — ideal para quem quer organizar as finanças de forma prática.

---

## 🚀 Funcionalidades

- 👤 Cadastro de usuário (nome personalizado)
- 💾 Persistência de dados com localStorage
- 📅 Controle de gastos por mês
- 💰 Registro de valor recebido mensal
- ➖ Cálculo automático do saldo restante
- 📊 Gráfico de gastos por categoria
- 🗂️ Categorias de despesas (Alimentação, Transporte, etc.)
- 🌙 Interface moderna com dark mode
- 📱 Responsivo (funciona bem no celular)

---

## 🧠 Como funciona

- Cada usuário tem seus próprios dados
- Cada mês armazena:
  - lista de gastos
  - valor recebido
- Os dados são salvos automaticamente no navegador (localStorage)
- Ao trocar de mês ou usuário, os dados são carregados automaticamente

---

## 🛠️ Tecnologias utilizadas

- React

- Vite
- Tailwind CSS
- Recharts (gráficos)
